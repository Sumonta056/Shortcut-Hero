#### ✍️ Open Tasks View

```bash
Windows key  + Tab
```

#### ✍️ Display and hide the desktop

```bash
Windows key  + d
```

#### ✍️ Switch between open apps

```bash
Alt + Tab
```

#### ✍️ Open the Quick Link menu

```bash
Windows key  + x
```

#### ✍️ Lock your PC

```bash
Windows key  + l
```

#### ✍️ Rename the selected file or folder

```bash
F2
```

#### ✍️ Open the selected folder properties

```bash
Alt + Enter
```

#### ✍️ Undo Items

```bash
Ctlr + z
```
#### ✍️ Redo Items

```bash
Ctlr + y
```


#### ✍️ Open Start Menu

```bash

Ctrl+Escape
```

#### ✍️ Open Settings

```bash
Ctrl+,
```

#### ✍️ Run Code

```bash
Ctrl+Alt+N
```

#### ✍️ Format Code Style

```bash
Shift+Alt+F
```

#### ✍️ Open Terminal

```bash
Ctrl+`

Shift+Ctrl+` # Create a new terminal
```

#### ✍️ Toggle Sidebar

```bash
Ctrl+B
```

#### ✍️ Open New Window/Instance

```bash
Shift+Ctrl+N
```

#### ✍️ Close Window

```bash
Ctrl+W
```

#### ✍️ Sidebar Focus & Use Sidebar Files

```bash
Shift+Ctrl+E
```

#### ✍️ Quick File Open From Current Directory

```bash
Ctrl+P
```

#### ✍️ Open File From Explorer

```bash
Ctrl+O
```

#### ✍️ Reopen Close Files

```
Shift+Ctrl+T
```

#### ✍️ Zoom IDE

```bash
Ctrl + + # Zoom in

Ctrl + - # Zoom out
```
